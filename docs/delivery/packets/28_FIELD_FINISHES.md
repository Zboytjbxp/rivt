# Packet 28 - Field Finishes

## Goal

Make appearance a real RIVT product preference rather than a row of
tool-adjacent color chips.

## Research Basis

The review of professional tool systems found a consistent product principle:
one restrained base, one high-contrast working accent, durable dark chrome,
and a recognizable system across surfaces rather than an isolated paint color.
RIVT applies that principle with original names and colors. It does not claim a
relationship with, or reproduce the branded trade dress of, tool manufacturers.

## Scope

- Rename the visual system from generic palettes to original RIVT Field
  Finishes: RIVT Standard, Signal Brass, Forge Red, Workshop Green,
  Copperline, and Harbor Blue.
- Let each finish control the canvas, surfaces, borders, top bar, navigation,
  active state, primary action, and supporting information tone together.
- Keep the Account panel compact with a current-finish summary and a single
  Customize appearance action.
- Open a full, scrollable Field Finishes editor from that action; retain the
  same editor in Settings.
- Add mobile smoke coverage proving the editor opens, exposes six choices,
  updates application chrome immediately, and has no horizontal overflow.

## Non-goals

- No manufacturer logos, names, protected visual signatures, or affiliation
  claims in the product.
- No fake Pro gate. Paid appearance entitlements require a server-enforced
  billing decision before this UI is restricted.
- No account, authentication, or persistence changes beyond the existing
  device-local appearance preference.

## Acceptance

- Selecting a Field Finish changes more than the main action color without a
  page reload.
- Account no longer embeds a long theme editor in the drawer.
- The compact Account entry and the full Settings editor identify the selected
  finish by name and remain usable on a phone.
- All appearance choices are explicitly identified as original RIVT finishes.
